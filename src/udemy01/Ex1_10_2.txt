以下、どのようなデータが画面に表示されるでしょう？
※プログラムで実行はしないで、紙とペンだけで考えましょう。

	String display = "";
	for(int i = 1 ; i <= 9  ; i++){
		display = "";
		for(int j = 1 ; j <= 9  ; j++){
			display = display + i*j + " ";
		}
		System.out.println(display);
	}
